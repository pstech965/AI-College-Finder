# 🎓 AI College Finder

AI College Finder is a smart, AI-powered application that helps students discover the best colleges based on their preferences such as budget, courses, location, and placements.

Using advanced AI models like Google Gemini, the app understands natural language queries and provides personalized, easy-to-read college recommendations instead of raw database results.

## ✨ Features
- 🔍 AI-based college search
- 🎯 Personalized recommendations
- 💰 Budget & placement focused results
- 📍 Location-based suggestions
- 📱 Clean, user-friendly output

## 🚀 Use Cases
- Find affordable colleges
- Discover top colleges for specific courses
- Get placement-focused recommendations

## 🛠 Tech Stack
- AI: Google Gemini  
- Frontend: React  
- Backend: Node.js  

## 📌 Goal
To simplify college selection using AI and smart recommendations.
